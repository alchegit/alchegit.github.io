- 마시멜로 슝 공방 | Defold | 약점 지지대를 맞춰 과자 탑을 무너뜨리는 슬링샷 물리 게임

비공개/폐쇄 유지
- 별점 모찌 상점, 별사탕 군집전, 비스킷 해양 조약, 포근한 스위치 정류장은 공개 카드에서 제외
- 다시 만들지 않을 규칙: 보는 맛만 있고 플레이어의 선택, 보상, 실패 이유가 약한 게임은 공개하지 않는다.
- 재미 보강 규칙: 첫 10~30초 안에 유저가 일부러 다시 시도하고 싶은 작은 고급 행동을 반드시 하나 넣는다.

새로 구현된 다크메이즈 기록
- 경로: games/dark-maze-lab-run/
- 입력: 키보드 없음, 캔버스 드래그와 조명 버튼만 사용
- 재사용 자산: lab-rat-walk-top 프레임, maze-floor, maze-wall, maze-entry, item-light-bulb
- 제거한 것: 광고 CTA, 스토어 설치 유도, 외부 광고 API, 로그인, 랭킹, 서버 저장
- 2026-07-07 수정: 캔버스 CSS가 와이드 영역을 허용해 오른쪽 빈 공간이 생기던 문제를 정사각형 보드로 고정, 모바일 제목/설명/팝업 잘림을 줄바꿈 구조와 폭 조정으로 수정
- 2026-07-07 추가 수정: 데스크톱/브라우저 확대 상태에서 2열 히어로가 점수 카드를 오른쪽으로 밀고 제목과 팝업이 과대해지던 문제를 단일 컬럼, 축소된 타이포, 높이 기준 보드 크기 제한으로 수정
- 2026-07-07 시야 규칙 수정: 전체 미로를 어둡게 보여주는 방식 금지, 조명 밖은 완전 암흑 처리, 캐릭터 주변 조명 반경 안의 벽/길 윤곽만 표시, 출구는 조명 반경 안에서만 노출
- 2026-07-07 플레이 감각 수정: 쥐 방향을 전진 방향으로 보정, 기본 조명 반경을 약 3분의 2로 축소, 벽/바닥 윤곽 구분 강화, 짧은 형광 발자국 trail 추가, 상단 재생 버튼으로 시작 웹툰 5장 보기 추가
- 2026-07-07 발자국/타일 보정: 벽과 바닥은 과하게 밝히지 않고 색온도와 선 두께만 분리, 발자국은 이미지 덩어리 대신 작은 발바닥+세 발가락 Canvas 도형으로 직접 렌더
- 재미 보강: 벽 충돌 파티클, 조명 사용 연출, 희미한 이동 흔적, 출구 근처 빛 신호로 목표감과 손맛을 추가
- 남은 개선 후보: 출구 근처 빛 맥박, 벽 충돌 피드백, 이전 최단 경로 발자국 연습 모드
